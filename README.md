# Getting Started

## Requirements:

```
Git: 2.47.1
Spring Boot: 3.4.1
Maven: 3.9+
Java: 17
Docker Desktop(Optional): Tested on 4.36.0
```

### Clone this repository:

```bash
git clone https://github.com/deepaksorthiya/spring-boot-3-security-magic-link.git
```

```bash
cd spring-boot-3-security-magic-link
```

### Start Docker:

```bash
docker compose up
```

### Build Project:

```bash
./mvnw clean package
```

### Run Project:

```bash
./mvnw spring-boot:run
```
